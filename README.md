# Sistema de Login de Funcionários (Python)

## Descrição

Este projeto consiste em um sistema simples em Python para cadastro de funcionários e validação de acesso (login) com base em nome e identificação numérica.

O objetivo é praticar conceitos fundamentais de programação, como variáveis, entrada de dados e estruturas condicionais.

## Funcionalidades

* Cadastro de três funcionários (nome e ID numérico)
* Validação de acesso por meio de nome e ID
* Retorno de mensagens de acordo com a autenticação

## Como funciona

1. O sistema solicita o cadastro de três funcionários.
2. Em seguida, solicita que o usuário informe seu nome e ID.
3. Os dados informados são comparados com os registros cadastrados.
4. O sistema informa se o acesso foi permitido, negado ou se o usuário não foi encontrado.

## Como executar

1. Certifique-se de que o Python está instalado em sua máquina.
2. Faça o download ou clone este repositório.
3. Execute o arquivo no terminal:

```bash
python nome_do_arquivo.py
```

## Estrutura do projeto

* `employee_login_system.py`: versão em inglês
* `sistema_login_funcionarios.py`: versão em português

## Objetivo

Projeto desenvolvido para fins educacionais, com foco na prática de lógica de programação e organização de código.

# Employee Login System (Python)

## Description

This project consists of a simple Python system for employee registration and login validation based on name and numeric identification.

The purpose is to practice fundamental programming concepts such as variables, user input, and conditional structures.

## Features

* Registration of three employees (name and numeric ID)
* Access validation using name and ID
* Output messages according to authentication results

## How it works

1. The system requests the registration of three employees.
2. It then prompts the user to enter their name and ID.
3. The provided data is compared with the registered records.
4. The system informs whether access is granted, denied, or if the user is not found.

## How to run

1. Ensure that Python is installed on your machine.
2. Download or clone this repository.
3. Run the file in the terminal:

```bash
python file_name.py
```

## Project structure

* `employee_login_system.py`: English version
* `sistema_login_funcionarios.py`: Portuguese version

## Purpose

This project was developed for educational purposes, focusing on practicing programming logic and code organization.
